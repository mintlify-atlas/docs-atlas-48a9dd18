## Built With Modern Technology

Built on **Next.js 16** with **React 19**, **Supabase** for backend services, and styled with **Tailwind CSS** and **shadcn/ui** components. The system leverages **FullCalendar** for interactive scheduling and **TanStack Table** for powerful data views.
